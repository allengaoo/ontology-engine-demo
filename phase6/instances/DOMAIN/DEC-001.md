## 背景

生产环境多次出现 500 响应直接包含 Python stacktrace，导致信息泄露并干扰客户端解析。

## 决策

所有 API 层异常统一包装为 `ErrorEnvelope`，字段：`code`（业务错误码）、`message`（用户可读）、`trace_id`（用于日志关联）。禁止 `str(exc)` 直接返回。

## 备选

各模块自定义错误结构（已否决：响应格式不一致，客户端难以统一处理）。
